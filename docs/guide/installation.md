# 安装

### 环境支持

由于 Vue 3 不再支持 IE11，Storm-UI 也不再支持 IE 浏览器。

### 使用包管理器

我们建议您使用包管理器（如 NPM、Yarn 或 pnpm）安装 Storm-UI。

```shell
# 选择一个你喜欢的包管理器

# NPM
$ npm install storm-ui --save

# Yarn
$ yarn add storm-ui

# pnpm
$ pnpm install storm-ui
```

### 浏览器直接引入

直接通过浏览器的 HTML 标签导入 Storm-UI，然后就可以使用全局变量 StormUI 了。
